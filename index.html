Project Setup Guide

Welcome to the ultimate guide on how to kickstart your web project using Antigravity AI.

Step 1: Create a Folder

First, create an empty folder on your computer where your project will live. For example, you can name it my-new-project.

Step 2: Open in Antigravity

Next, open the Antigravity application. Drag and drop the folder you just created into Antigravity, or use the "Open Folder" option to load your workspace.

Step 3: Start Building

Now, simply ask Antigravity to build what you want! For example, you can type: "Create a complete web project step-by-step" and watch the magic happen.

Step 4: Upload to GitHub

Go to GitHub and follow these exact steps to upload your project:

  1. Log in to your account at github.com.
  2. Click the "+" icon in the top right corner and select "New repository".
  3. Name your repository (e.g., my-simple-website), leave it as Public, and click the green "Create repository" button.
  4. On the next page, click the blue link that says "uploading an existing file".
  5. Open your local folder (d:\steps of project\my-simple-website), select all files, and drag them into the upload box on GitHub.
  6. Type a commit message like "Initial commit" at the bottom and click "Commit changes".

Step 5: Deploy on Vercel

Log in to Vercel, click "Add New Project", and import your newly created GitHub repository. Once deployed, copy the provided link and paste it into your browser to see your live website!
